# Selected Concepts for the Phase 2 Pilot

The first pilot study will investigate the following six concepts:

1. Hand anatomy
2. Facial consistency
3. Text readability
4. Repeated texture patterns
5. Object boundary consistency
6. Background coherence

These concepts were selected because they are human-understandable, visually diverse, and potentially suitable for both manual annotation and automatic estimation.

At this stage, they are hypotheses rather than confirmed indicators of synthetic imagery.
