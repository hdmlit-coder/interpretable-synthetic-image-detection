# CIFAKE Baseline

## Paper and Repository

CIFAKE provides baseline image classifiers for experiments involving real and AI-generated images.

## Objective

The purpose of this reproduction was to establish a first CNN-based experimental baseline and verify the complete workflow for dataset preparation, model training, evaluation, and result reporting.

## Architecture

The reproduced experiment uses a LeNet convolutional neural network.
